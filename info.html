Just some very important information

The site only checks for the latest message every 5 seconds for the beta and every 10 seconds for the main chat. This means that if you send a message and somebody else sends another one just after yours, your message might not be displayed to other people. In the beta this is less of a problem but in the main chat, please wait until another message pops up before sending yours.

There is a swear filter. Although if some users are bypassing the filter and/or being rude, I might not be online at times, so please email EnZon3@protonmail.com if someone is bullying you.

Also note that usernames, except for mine are completely ununiqe so, anybody can have the same username.
